# Operators Perform Actions On Values

Operators are symbols we use to act on
variables and values. He are some examples:

## Arithmetic Operators
|Operator|Action|
|--------|------|
|+| Addition|
|-| Subtraction|


## Assignment Operators
|Operator|Action|
|--------|------|
|= | Sets a variable to a value|

## Relational Operators
|Operator|Action|
|--------|------|
|== | *Is equal to* |
|!= | *Is not equal to* |
|<| *less than* |
|>| *greater than* |

## Logical Operators
|Operator|Action|
|--------|------|
|&&| True only if all expressions are true|
|\|\| | True if at least one expression is true|
