# Object Oriented Programming

<TimeStamp updated="February 27, 2022, 9:20  PM ET"/>

OOP is a paradigm where the main _actors_ are
_objects_. Objects come from classes. They are
concrete instances of the class. There are also
common design goals and principles associated 
with OOP.
