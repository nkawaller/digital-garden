# Passing Arguments To Functions

<TimeStamp updated="February 27, 2022, 9:20  PM ET"/>

[Functions](/what-is-a-function) take in data, transform it,
and return it. The way we pass in data is through arguments. 
We can pass arguments by [value](/passing-arguments-by-value)
or by [reference](/passing-arguments-by-reference).

In Java we call arguments _parameters_.
