# Represent An Excel Spreadsheet As JSON

<TimeStamp updated="February 27, 2022, 9:20  PM ET"/>

If we have an Excel spreadsheet containing 
rows and columns of data, we can also represent that data
in a list of JSON objects. 

* Each row will be a separate object
* The column names will be the keys
* The data in each cell maps to its key
