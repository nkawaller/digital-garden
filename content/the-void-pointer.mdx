# The Void Pointer

<TimeStamp updated="February 27, 2022, 9:20  PM ET"/>

Usually, when we declare [pointers](/what-are-pointers),
we need to indicate the [type](/what-is-a-data-type) of 
variable it points to. But the `void` pointer can point to
a variable of any type.
