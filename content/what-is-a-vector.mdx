# What Is A Vector?

<TimeStamp updated="February 27, 2022, 9:20  PM ET"/>

In C++, a `vector` is a data structure that's similar
to an [array](/what-is-an-array). Vectors can be easier 
to use though, because they provide run-time index 
checking, and they can be dynamically resized.
