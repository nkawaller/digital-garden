# Example Pointer Scenario

Working with [pointers](/what-are-pointers) takes
practice. Here's an 
example scenario.

> Let's say `q` is a pointer to an `int`,
> and we set `q = &x`
* We can get the value of `x` with `*q`
* Also, assigning an integer value to `*q` would change
the value of `x`

Check out more examples [here](/working-with-pointers).
