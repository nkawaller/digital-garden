# Designing A C++ Program

<TimeStamp updated="February 27, 2022, 9:20  PM ET"/>

We should consider a few things when designing a 
[computer program](/what-is-a-computer-program):
- how to split our program into [classes](/what-is-a-class)
- how those classes will interact
- what [data](/what-are-class-members) each class will store
- what actions each class will perform
