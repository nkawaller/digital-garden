# Fundamental Data Types In C++?

C++ has some data types built into the language. Here's 
a list of some of the most commonly used ones.

| Type | Size | Represents |
|:------|:------|:------------|
|`bool`|8 bits| true/false values|
|`char`| 8 bits| a single character|
|`int`|16 bits | basic integer type|
|`short`| 16 bits| short integer|
|`long`|32 bits| long integer |
|`float`| 16 bits |single-precision floating point|
|`double`|32 bits |double-precision floating point|
|`enum`| ? | user-defined set of constants|
