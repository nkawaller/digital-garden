# What Are HTTP Headers?

<TimeStamp updated="February 27, 2022, 9:20  PM ET"/>

When the server and client want to send extra
information in thier requests and responses, they 
use HTTP headers. Headers are structured as key-value 
pairs and can be used for a variety of things
such as authorization, setting the viewport width, 
and sending [cookies](/what-are-http-cookies).
